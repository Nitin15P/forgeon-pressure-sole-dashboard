# ForgeOn Pressure Sole Dashboard

A React dashboard for visualizing pressure sole analytics for fast bowling.

## Deployment

This repository is configured for Vercel deployment. The dashboard is located in the `pressure-sole-dashboard` folder.

### Vercel Setup

When importing this repo to Vercel:

1. **Root Directory**: Set to `pressure-sole-dashboard`
2. **Framework Preset**: Vite (auto-detected)
3. **Build Command**: `npm run build`
4. **Output Directory**: `dist`

## Local Development

```bash
cd pressure-sole-dashboard
npm install
npm run dev
```

## Build

```bash
cd pressure-sole-dashboard
npm run build
```
